# Zest - my personal blog

## Development

### installing dev tools
1.  install Ruby 2.7
2. install bundler
```
gem install bundler
```
3. install Ruby gems
```
cd site
bundle install
```

### Making changes to the site
launching a local interactive version of the site in the browser: 

```
cd site 
bundle exec jekyll serve --livereload --open-url
```
